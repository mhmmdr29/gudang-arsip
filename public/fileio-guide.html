📤 Upload ke File.io

Ikuti langkah-langkah untuk upload file ZIP ke File.io

1
Buka File.io

Buka browser dan akses: https://file.io

2
Login / Daftar

Login dengan akun Anda atau daftar gratis di File.io

3
Akses File dari Server

Buka link ini di browser (untuk mendownload file ke laptop Anda dulu):

📥 Download: gudang-arsip.zip

Setelah selesai didownload, file ZIP ada di folder Downloads laptop Anda.

Ingat lokasi file!

4
Upload ke File.io

Kembali ke File.io (tab browser yang sudah dibuka)

Klik tombol "Upload" atau "+"

Pilih file gudang-arsip.zip dari folder Downloads

Tunggu proses upload selesai

5
Share Link

Setelah upload selesai, File.io akan memberikan link sharing

Klik tombol "Copy Link" atau "Share"

Link sharing bisa Anda bagikan ke orang lain!

⚠️ CATATAN: File.io menyimpan file selama 14 hari pada akun gratis
6
Download Link (Untuk Orang Lain)

Orang lain bisa mendownload dengan link sharing yang sudah Anda copy.

Mereka bisa download dan ikuti langkah instalasi yang sama.

7
Cara Instalasi (Setelah Didownload)

1. Ekstrak ZIP: Right-click → Extract All

2. Install Bun/Node.js: Download dari bun.sh atau nodejs.org

3. Install Dependencies: bun install

4. Jalankan: bun run dev

5. Buka: http://localhost:3000
